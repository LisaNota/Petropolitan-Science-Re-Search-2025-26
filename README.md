# Petropolitan-Science-Re-Search-2025-26

Данный репозиторий представляет собой решение 3х задач отборочного этапа олимпиады по направлению "Вычислительные технологии"

Полное решение, включая код, комментарии, описания, а также отчёт, содержащий анализ, выводы и результаты, находятся в папках, соответствующих номеру задания.

В каждой папке task_i, i=1,2,3:
    readme.md - описание задачи, содержание папки. необходимые уточнения и описания, если они необходимы спецификой задачи
    solution.ipynb - ход решения задачи, объяснение рассуждений, промежуточные шаги, формирование результата
    main.py - файл, воспроизводящий итоговое решение из solution.ipynb
    src/* - папка с дополнительными файлами-модулями (при необходимости). Каждая функция содержит докстринги и описания
    report.docx - отчет, содержащий полное описание проделанной работы (включая введение, анализ, решение, блок-схемы, тесты, и другую доп информацию, требуемую конкретным заданием)
