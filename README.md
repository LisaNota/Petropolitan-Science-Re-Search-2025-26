# Petropolitan-Science-Re-Search-2025-26

Данный репозиторий содержит решения **трёх задач отборочного этапа** олимпиады *Petropolitan Science (Re)Search* по направлению **«Вычислительные технологии»**.

Полное решение каждой задачи (код, комментарии, описания, промежуточные шаги и итоговые результаты) находится в папках, соответствующих номеру задания.

---

## Структура репозитория

Репозиторий организован по папкам заданий:

```
task_1/
task_2/
task_3/
```

В каждой папке `task_i`, где `i = 1, 2, 3`, находятся:

- `readme.md` — описание задачи, содержание папки, анализ методов, описание решение, сравнение достоинств и недостатков, а также другие необходимые выводы (если это требуется спецификой задания)
- `solution.ipynb` — полный ход решения: рассуждения, промежуточные шаги, визуализации (если применимо), формирование результата
- `main.py` — файл, воспроизводящий итоговое решение из `solution.ipynb`
- `src/` — папка с дополнительными модулями (при необходимости)  
  *(каждая функция внутри модулей содержит докстринги и описания)*

---

## Как пользоваться решениями

### 1) Быстрый запуск (воспроизведение результата)
Для получения итогового результата по конкретной задаче достаточно выполнить `main.py` в папке задания:

```bash
cd task_i
python main.py
```

### 2) Полный разбор решения
Если требуется увидеть ход решения и все промежуточные шаги, откройте ноутбук:

- `task_i/solution.ipynb`

и выполните ячейки последовательно.

- Python 3.10+ (рекомендуется)
- Зависимости могут отличаться от задачи к задаче и, при необходимости, описаны в `readme.md` конкретного задания.
